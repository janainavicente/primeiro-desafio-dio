Como surgiu a internet?

A internet surgiu a partir de pesquisas militares no auge da Guerra Fria. Neste contexto, em que os dois blocos ideológicos e politicamente antagônicos exerciam enorme controle e influência no mundo, qualquer mecanismo, qualquer inovação, qualquer ferramenta nova poderia contribuir nessa disputa liderada pela União Soviética e pelos Estados Unidos: as duas superpotências compreendiam a eficácia e a necessidade absoluta dos meios de comunicação. Nessa perspectiva, o governo dos Estados Unidos temia um ataque russo às bases militares. Um ataque poderia trazer a público informações sigilosas, tornando os EUA vulneráveis. Então foi idealizado um modelo de troca e compartilhamento de informações que permitisse a descentralização das mesmas. Assim, se o Pentágono fosse atingido, as informações armazenadas ali não estariam perdidas. Era preciso, portanto, criar uma rede, a ARPANET, criada pela ARPA, sigla para Advanced Research Projects Agency

Conceito de HTML

HTML é uma linguagem de marcação que define a estrutura do seu conteúdo. HTML consiste em uma série de elementos, que você usa para incluir, ou encapsular, diferentes partes do conteúdo para fazê-lo aparecer de uma determinada maneira, ou agir de uma certa maneira

O que são tags HTML?

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página

A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”).

As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento. As tags que precisam de fechamento possuem a sintaxe , já as que não precisam de fechamento possuem como estrutura

Estrutura do HTML

Para utilizar as Tags HTML e construir uma boa página web você precisa da estrutura base para iniciar o trabalho.

  • Html:5 - Informa ao navegador qual o tipo de linguagem utilizada.

  • Head - É o cabeçalho do seu código

Listas

As listas são classificadas em: Ordenadas e não ordenadas.

Nas listas ordenadas utilizamos a tag ol, porém para elaborar uma lista temos que usar uma tag filha, funciona como uma tag dentro da outra. A tag utilizada dentro da tag ol é a tag li. Listas ordenadas são numeradas. Exemplo: 1,2,3,4 ...

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Nas listas não-ordenadas utilizamos a tag ul, porém para elaborar uma lista temos que usar uma tag filha, funciona como uma tag dentro da outra. A tag utilizada dentro da tag ul é a tag li. Listas não ordenadas temos marcadores.

Exemplo:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Links

Utiliza- se a tag a para colocar um link na sua página ou texto, que levará o usuário a outra página. Você pode colocar esse link em listas também. Para definir onde o link irá abrir é utilizado um atributo, nessa teag utilizamos o atributo href. Você pode ter um página abrindo na mesma página, onde utilizamos o atributo target"blank" ou abrir em outra página, utilizando o atributo target"self".

Para voltar para a página que estava pode ser feito um novo link com o endereço da pagina anterior ou arquivo dentro do href.

Trabalhando com formulários

A tag utilizada para trabalhar com o formulário é a tag: Form.

O elemento HTML form representa uma seção de um documento que contém controles interativos que permitem ao usuário submeter informação a um determinado servidor web.

As informações do formulario serão colocadas dentro da tag form, e nomeadas através do atributo input, onde colocamos o tipo da informação.

Ao final do formulário temos que colocar um botão para enviar os dados para seu destino, o botão é criado através da tag button é necessário definir qual tipo de button será, isso é definido pelo atributo type, sendo que ele pode ser submit, reset, button.

O comportamento padrão do botão. Os valores possíveis são:

  • submit:O botão envia os dados do formulário para o servidor. Esse é o padrão se o atributo não for especificado para botões associados a um , ou se o atributo for um valor vazio ou inválido.
  • reset:O botão redefine todos os controles para seus valores iniciais, como input type="reset".
  • button:O botão não tem comportamento padrão e não faz nada quando pressionado por padrão. Ele pode fazer com que os scripts do lado do cliente ouçam os eventos do elemento, que são acionados quando os eventos ocorrem.
